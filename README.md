# Instagram Clone
Instagram clone with MERN stack

## Notice ⭕️
Please check "Client" branch for React and "Server" branch for Node

## Usage
To run this you need to have Mongodb installed on your machine.
Clone project then 
```bash
git checkout Client
npm install
```
and run client with 
```bash
npm start
```
For server 
```bash
git checkout Server
npm install
```
and fire up the Server with
```bash
nodemon app
```
